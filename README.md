# Object Detection Using TensorFlow

## 📌 Overview
This project implements an Object Detection system using TensorFlow and Deep Learning techniques to identify and detect objects from images with high accuracy.

---

## 🚀 Features
- Real-time object detection
- Multiple object recognition
- Bounding box visualization
- TensorFlow-based deep learning model
- Image prediction and classification

---

## 🛠 Technologies Used
- Python
- TensorFlow
- OpenCV
- NumPy
- Matplotlib
- Jupyter Notebook
